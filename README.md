# BOJ Step Coach - Claude Code Plugin

백준 온라인 저지(BOJ) 단계별 문제풀이를 **올해 안에 클리어**하는 것을 목표로, 체계적인 학습 코칭을 제공하는 Claude Code 플러그인입니다.

문제 번호만 입력하면 풀이 환경이 자동으로 세팅되고, 연간 계획 수립부터 진도 추적, 단계별 힌트, 코드 리뷰, 반례 분석까지 문제풀이의 전 과정을 지원합니다.

---

## 주요 기능

- **파일 자동 생성**: 문제 번호만 입력하면 solution + test 코드 생성
- **연간 학습 계획**: 올해 안에 단계별 문제풀이 클리어 목표 플랜
- **진도 추적**: 현재 위치, 페이스 체크, 다음 문제 추천
- **단계적 힌트**: 정답을 바로 주지 않고 5단계 힌트 시스템
- **코드 리뷰**: 시간/공간 복잡도 분석, 개선 제안
- **반례 찾기**: 경계값, 특수 케이스, 함정 케이스 자동 생성
- **자동 업데이트**: 문제 완료 시 plan.json, progress.json, README 자동 갱신

### 단계적 힌트 시스템

| 레벨 | 내용 |
|------|------|
| Lv.1 | 방향 제시 — 핵심 인사이트를 질문으로 유도 |
| Lv.2 | 알고리즘 공개 — 사용할 알고리즘 이름과 개념 설명 |
| Lv.3 | 접근법 설명 — 의사코드 수준의 풀이 전략 |
| Lv.4 | 핵심 코드 조각 — 가장 어려운 부분만 3~5줄 제공 |
| Lv.5 | 전체 풀이 — 3회 이상 요청 시에만, 한글 주석 포함 |

### 지원 언어

Python, TypeScript, C++, Java, JavaScript, Kotlin

## 설치 방법

Claude Code 터미널에서:

```bash
# 1. 마켓플레이스 추가
/plugin marketplace add min-99/boj-step-coach

# 2. 플러그인 설치
/plugin install boj-step-coach@boj-step-coach
```

---

## 사용법

설치 후 Claude Code에서 자연어로 말하면 자동 활성화됩니다:

```
# 새 문제 시작
"백준 1000번 풀자"
"#15649 파일 만들어줘"

# 계획 및 진도
"올해 계획 세워줘"
"진도 알려줘"
"잘 하고 있어?"

# 도움 요청
"모르겠어, 힌트 줘"
"어떻게 접근하면 돼?"

# 풀이 완료 후
"다 풀었어, 리뷰해줘"
"맞는지 확인해줘"

# 오답 디버깅
"틀렸어, 반례 찾아줘"
"왜 안 되는지 모르겠어"
```

## 프로젝트 구조

플러그인이 기본으로 제안하는 풀이 디렉토리 구조입니다. 기존 프로젝트에 풀이가 있으면 그 패턴을 자동으로 감지하여 따릅니다.

```
boj/
├── README.md                       # 전체 진도 요약 (자동 업데이트)
├── progress.json                   # 문제별 풀이 기록
├── plan.json                       # 연간 학습 계획
├── step01_입출력과_사칙연산/
│   ├── p1000_A+B/
│   │   ├── solution.py             # 풀이 코드 (사용자 작성)
│   │   └── test_solution.py        # 테스트 코드 (자동 생성)
│   └── ...
├── step02_조건문/
│   └── ...
└── ...
```

## 설계 철학

| 원칙 | 설명 |
|------|------|
| **정답을 주지 않는다** | 5단계 힌트 시스템을 통해 스스로 풀도록 유도합니다 |
| **자동화로 마찰을 줄인다** | 파일 생성, 테스트, 진도 기록을 자동화해 풀이에만 집중하게 합니다 |
| **현실적인 목표** | 난이도와 기간을 고려한 주간 계획으로 번아웃 없이 완주를 돕습니다 |
| **기존 프로젝트 존중** | 언어와 구조를 강요하지 않고 사용자의 기존 패턴을 감지해 따릅니다 |

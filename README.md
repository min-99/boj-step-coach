# BOJ Step Coach — Claude Code Skill

백준 온라인 저지(BOJ) 단계별 문제풀이를 **연간 클리어 목표**에 맞춰 체계적으로 코칭하는 Claude Code 스킬입니다.

---

## 주요 기능

### 1. 풀이 파일 자동 생성
문제 번호를 입력하면 즉시 시작할 수 있는 파일을 만들어 줍니다.

- acmicpc.net에서 문제 정보(제목, 제한, 예제)를 자동으로 가져옵니다.
- 풀이 템플릿(`solution.py` 등)과 테스트 코드(`test_solution.py`)를 함께 생성합니다.
- 예제 입출력과 엣지 케이스가 테스트에 자동으로 포함됩니다.
- 기존 프로젝트 구조와 언어를 자동으로 감지해 패턴을 맞춥니다.
- 지원 언어: **Python, TypeScript, C++, Java, JavaScript, Kotlin**

### 2. 연간 학습 계획 생성
"계획 세워줘"라고 말하면 됩니다.

- 오늘부터 12월 31일까지 남은 기간을 기반으로 주간 계획을 수립합니다.
- 단계별 난이도를 반영해 초반은 빠르게, 후반은 여유 있게 배분합니다.
- 버퍼 주간을 포함해 실현 가능한 목표를 제시합니다.
- `plan.json`으로 저장되어 진도와 항상 연동됩니다.

### 3. 진도 추적 및 피드백
"잘 하고 있어?" "진도 알려줘"로 현재 상태를 확인합니다.

```
📊 진도 현황

🗓️ 현재: 2026년 4월 9일 (15주차)
📈 전체 진행률: ██████░░░░ 62% (186/300문제)

이번 주 목표
- 단계: step15_백트래킹
- 목표 문제: 5문제 중 3문제 완료 ✅

컨디션 체크
- ✅ 계획 대비 1일 앞서가고 있습니다! 페이스 좋아요.
- 💡 평균 풀이 시간: 45분/문제
```

문제를 풀 때마다 `progress.json`과 `README.md`(진도표)가 자동으로 업데이트됩니다.

### 4. 단계적 힌트 시스템
"모르겠어", "힌트 줘"라고 말하면 정답 없이 단계별로 도움을 줍니다.

| 레벨 | 내용 |
|------|------|
| Lv.1 | 방향 제시 — 핵심 인사이트를 질문으로 유도 |
| Lv.2 | 알고리즘 공개 — 사용할 알고리즘 이름과 개념 설명 |
| Lv.3 | 접근법 설명 — 의사코드 수준의 풀이 전략 |
| Lv.4 | 핵심 코드 조각 — 가장 어려운 부분만 3~5줄 제공 |
| Lv.5 | 전체 풀이 — 3회 이상 요청 시에만, 한글 주석 포함 |

### 5. 코드 리뷰
"다 풀었어", "리뷰해줘"라고 말하면 테스트 실행 후 리뷰를 제공합니다.

- 시간/공간 복잡도 분석
- 잘한 점 피드백
- 가독성·효율성·안전성 관점의 코드 개선 조언 (개선 전/후 비교 포함)
- 핵심 정리 및 유사 문제 추천

### 6. 반례 자동 생성
"틀렸어", "반례 찾아줘"라고 말하면 코드를 분석해 놓친 케이스를 찾아줍니다.

- 경계값, 특수 케이스, 함정 케이스를 체계적으로 생성합니다.
- 실패 케이스만 보여주고 정답은 알려주지 않아 스스로 원인을 찾도록 유도합니다.
- 발견한 반례를 테스트 파일에 자동으로 추가합니다.

## 설치 방법

Claude Code 프로젝트 디렉토리에서:

```bash
claude skill install boj-step-coach
```

또는 `.claude/skills/boj-step-coach/SKILL.md` 파일을 직접 추가합니다.

---

## 사용법

스킬이 설치된 상태에서 Claude Code와 자연어로 대화하면 됩니다.

```
# 새 문제 시작
"1000번 풀자"
"#15649 파일 만들어줘"

# 계획 및 진도
"올해 계획 세워줘"
"진도 알려줘"
"잘 하고 있어?"

# 도움 요청
"모르겠어, 힌트 줘"
"어떻게 접근하면 돼?"

# 풀이 완료 후
"다 풀었어, 리뷰해줘"
"맞는지 확인해줘"

# 오답 디버깅
"틀렸어, 반례 찾아줘"
"왜 안 되는지 모르겠어"
```

## 프로젝트 구조

스킬이 기본으로 사용하는 디렉토리 구조입니다. 기존 구조가 있으면 그 패턴을 그대로 따릅니다.

```
boj/
├── README.md                       # 전체 진도 요약 (자동 업데이트)
├── progress.json                   # 문제별 풀이 기록
├── plan.json                       # 연간 학습 계획
├── step01_입출력과_사칙연산/
│   ├── p1000_A+B/
│   │   ├── solution.py             # 풀이 코드 (사용자 작성)
│   │   └── test_solution.py        # 테스트 코드 (자동 생성)
│   └── ...
├── step02_조건문/
│   └── ...
└── ...
```

## 설계 철학

- **정답을 주지 않는다** — 힌트 5단계를 통해 스스로 풀도록 유도합니다.
- **자동화로 마찰을 줄인다** — 파일 생성, 테스트, 진도 기록을 자동화해 풀이에 집중하게 합니다.
- **현실적인 목표** — 난이도와 기간을 고려한 주간 계획으로 번아웃 없이 완주를 돕습니다.
- **기존 프로젝트 존중** — 언어와 구조를 강요하지 않고, 사용자의 패턴을 감지해 따릅니다.
